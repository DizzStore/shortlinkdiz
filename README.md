# ShortLink - Pemendek Tautan di Termux

Alat sederhana untuk memperpendek tautan URL langsung dari terminal Termux Anda.

Fitur

· ✅ Memperpendek tautan dengan cepat
· ✅ Mendukung multiple URL shortener services
· ✅ Tanpa iklan yang mengganggu
· ✅ Ringan dan mudah digunakan
· ✅ Bisa untuk pemula

Persyaratan

· Termux (terinstall dari F-Droid)
· Koneksi internet aktif

Instalasi
$ pkg update && pkg upgrade

$ pkg install git

$ pkg install python

$ git clone https://github.com/DizzStore/shortlinkdiz.git

$ cd shortlinkdiz

$ ls

$ python shortlinkdiz.py
